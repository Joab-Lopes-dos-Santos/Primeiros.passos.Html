Tags

(voltar)

Uma tag é um elemento utilizado em HTML para marcar e estruturar o conteúdo de uma página da web.

Ela é representada por códigos delimitados por sinais de menor e maior ("<" e ">") e serve para identificar diferentes elementos, como parágrafos, títulos, imagens, links, entre outros.

As tags podem ser de abertura e fechamento, envolvendo o conteúdo a ser marcado, ou podem ser autônomas, inserindo elementos independentes na página.

As tags são essenciais para a formatação e organização do conteúdo da página e permitem que os navegadores interpretem corretamente e exibam o conteúdo da web de forma adequada.

Atributos das Tags

(voltar)

Atributos de tags são informações adicionais que podem ser incluídas dentro das tags em HTML.

Eles fornecem características ou configurações específicas para os elementos marcados pelas tags.

Os atributos são usados ​​para controlar o comportamento, aparência ou funcionalidade dos elementos na página da web.

Alguns exemplos comuns de atributos são "src" (especifica a origem de uma imagem), "href" (define o destino de um link), "class" (atribui uma classe CSS ao elemento) e "alt" (fornece uma descrição alternativa para uma imagem).

Os atributos permitem personalizar e ajustar o comportamento dos elementos marcados pelas tags de acordo com as necessidades do desenvolvedor e os requisitos da página.

Textos

(voltar)

Em HTML, um texto é qualquer conteúdo que é exibido diretamente na página da web.

Ele pode ser uma palavra, uma frase, um parágrafo ou mesmo um conjunto de parágrafos até.

Os textos em HTML são inseridos dentro e elementos de marcação de texto, como as tags < p > para parágrafos,< h1 > a < h6 > para títulos de níveis diferentes, < span > para trechos de texto específicos, entre outros.

Listas Ordenadas

(voltar)

Listas ordenadas em HTML são elementos usados ​​para criar listas em que os itens são apresentados em uma ordem específica, geralmente numerados ou com símbolos.

As listas são definidas pela tag < ol > (lista ordenada) e cada item é marcado com uma tag < li > (item da lista).

Isso permite exibir os itens em sequência, facilitando a compreensão e organização do conteúdo para os usuários.
